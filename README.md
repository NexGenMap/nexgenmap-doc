# NexGenMap

This project aims to establish the next generation of land use and land cover mapping and monitoring tools by leveraging Planet’s improved satellite imagery and Google Earth Engine’s computing platforms. You can access the official documentation [here](https://nexgenmap.readthedocs.io)

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. See deployment for notes on how to deploy the project on a live system.
